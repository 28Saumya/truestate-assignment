# TruEstate Retail Sales Management System

A full-stack retail sales management system built for the TruEstate SDE Intern assignment.

## Overview
- Retail sales dashboard with KPIs and city-wise summary
- Transaction table with search, filters, sorting, and pagination
- Backend in Node.js + Express reading from large CSV dataset
- Frontend in React (Vite) with modular components

## Tech Stack
- Backend: Node.js, Express, csv-parser
- Frontend: React, Vite, Axios
- Data: CSV dataset provided in assignment
